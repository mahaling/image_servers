# Image Servers #
Image server modules for development of deep learning or image processing.

## Dependences ##
* Python2 (or 3)
* Flask
* Flask-Socketio
* OpenCV

## DNN Examples ##
To run DNN examples, please install chainer and
download `bvlc_googlenet.caffemodel` to `dnn_models` (see `dnn.py`).
